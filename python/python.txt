 Для того чтобы при импорте например, весь включаемый код не выполнился помещают следующую конструкцию в конец импорттируемого файла:

	if __name__ = '__main__':
	    main()
	Это можно использовать для отладки - дергать по одной какие-нить ф-ии из этого файла

Для того чтоб узнать какие методы есть у како-нить переменной можно воспользоваться встроенной переменной dir():
	
	a = 4
	dir(a)



						        
Альтернативная версия dir(он просто перечисляет все методы, без обьяснений) - функция help, по сути это ман страница для конкретного модуля/переменной/класса напрмер
	
	import sys
	help(sys)
	Далее чиатем перелистывая стрелочками подробное описание что делает каждая функция

Когда хотим чтоб все спецсимоволы отсались как есть(не экранировались) и печатались, например
	
	raw = r"Some string"
	print(raw)

Юникодная строка:

	st = u"Some string"

Байтовая строка

	st = b"Some string"

Оператор % для строк как на Си в printf:

	st = ("%d pretty %s" % (2,"chicks"))

При работе с модулями, часто обращаешься к эл-там модуля через имя модуля, а это не удобно, слишком много писать:
	import numpy
	numpy.eye.....
Чтоб избежать этого можно включить в наше пространство имен имя модуля:
	from numpy import numpy или
	from numpy import *

if 'a' in d:
	d['a'].append(1)
else:
	d['a'] = [1]
